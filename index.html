To test your programming skill, we'd like you to have a go at a small task similar to the kind of thing you could end up working on here.

We have provided you with a template below in div#application. There's a link to our API which provides the tile data stored in it's data-source attribute. There are three types of tiles: Quote panels, Manager portfolio panels and Multi Manager portfolio panels. The data provided is grouped by tile type which matches CSS class names on tiles in the template.

Your task is to use this data to produce an animated presentation that will showcase all content provided.

You should start with the presented tile set. A tile replacement should then occur every 1-3s.

Some things to note:
  • You may not use any JavaScript libraries (though testing frameworks, module systems and build processes are permitted and encouraged).
  • Your solution should work in any modern browser, but you don't need to worry about making it fully cross browser or supporting older browsers.
  • A single tile may never be updated twice in a row.
  • A single Manager portfolio, Multi Manager portfolio and Quote can never be shown twice in a row.
  • Tiles need to be updated at random.
  • Some managers have more than one portfolio, only one can be shown at a time for a single manager.

We hope you enjoy the task, and we look forward to seeing your solution. Feel free to get in touch if you have any questions at all about the task.
